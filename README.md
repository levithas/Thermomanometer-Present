# Thermomanometer
This program displays the current temperature and air pressure on an LED strip. Temperature is shown in Â°C and air pressure in hPa.

### Hardware
- ESP8266 (Microcontroller)
- BMP180 (Temperature and air pressure sensor)
- Neopixel LED strip (configured for 20 LEDs)
- 2 green LEDs

The microcontroller is powered by the build-in USB port.
